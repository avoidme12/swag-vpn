**SwagVPN** - это современный VPN-клиент на базе ядра **Xray**, обеспечивающий максимальную приватность и обход блокировок с использованием протоколов нового поколения.

---

## ✨ Основные фишки

* 🚀 **Ядро Xray/V2Ray:** Максимальная скорость и стабильность соединения.
* 📊 **Adaptive Monitoring:** Живой график скорости в секунду с адаптивным масштабированием.
* 🔐 **Encryption:** Все ключи конфигурации передаются в зашифрованном виде.
* 💳 **Smart Payment:** Быстрая оплата через СБП, карты и криптовалюту прямо в Telegram-боте.

---

## 📲 Как начать пользоваться?

1.  **Скачайте APK:** Перейдите во вкладку [Releases](https://github.com/avoidme12/swag-vpn/releases) и скачайте последнюю версию.
2.  **Получите ключ:** Зайдите в нашего [Telegram-бота](https://t.me/@sw4gvpnbot).
3.  **Активируйте:** Скопируйте ключ из бота и вставьте его в приложение.
4.  **Connect:** Нажмите на центральную кнопку и наслаждайтесь свободным интернетом!


---

## 🛠 Технологический стек

* **Frontend:** React Native (Android)
* **Core:** Xray-core / Tun2Socks
* **Backend (Bot):** Node.js + @mtcute + YooKassa
* **Protocol:** VLESS + Reality + XTLS-Reality-Vision
